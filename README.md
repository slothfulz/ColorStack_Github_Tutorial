Welcome to our GitHub crash course!  
Your mission: add your team name + fun fact to `scoreboard.md` and get your PR merged first!

Step 1: Fork this Repo
This will copy me repository into your github account. Think of this as making a copy a google doc you can't edit. 

Step 2: Clone This Repo into your local machine
```git clone https://github.com/andipandi1738/ColorStack_Github_Tutorial.git```
This will 

Step 3: Create  branch
```git checkout -b team-<name>```

Step 4: Edit ```scoreboard.md``` to add a line for your team

Use a code editor to edit the files

a. If you have VS Code you can open the file in there and save it
b. If you don't have that installed, you can still edit in your terminal using vim

Step 5: Push your code using standard git workflow
a. ```git add . ```
This will add all changes made to the staging area

b. ```git commit -m "add Team Taco to scoreboard"```
This will add commit the change to your local repository. All commits need a message.

c. ```git push```
This will push your code to the remote repo.
